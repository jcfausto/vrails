# vrails
A Vagrant Rails Environment Ready to Go! 

#How to use
1) Install vagrant if you don't have it yet.
2) git clone this repo
3) vagrant up

That's all! You are ready to go! 

#What is included by default
- Development tools
- Ruby
- Bundler
- Git
- SQLite
- PostgreSQL
- MySQL
- Nokogiri dependencies
- ExecJS runtime
